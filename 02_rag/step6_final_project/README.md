# Step 6: RAG 最终项目

## 🎯 项目目标

整合前面学到的所有知识，构建一个**生产级RAG问答系统**。

## 📋 项目功能

### 1. 多文档管理
- 支持导入多个文档
- 文档元数据管理
- 文档更新和删除

### 2. 智能检索
- 混合检索策略（向量 + 关键词）
- 重排序优化
- 上下文窗口管理

### 3. 完整的交互界面
- 命令行交互
- 流式输出
- 历史对话管理

### 4. 性能监控
- 检索耗时统计
- 生成耗时统计
- 相似度分析

