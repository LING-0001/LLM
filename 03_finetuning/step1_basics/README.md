## 💡 关键概念

### 预训练 vs 微调

```
预训练 (Pre-training):
- 在海量通用数据上训练
- 学习语言的基本规律
- 耗时长，成本高

微调 (Fine-tuning):
- 在特定领域的小数据集上训练
- 学习特定任务/风格
- 耗时短，成本低
```

### 为什么需要微调？

**场景1**: 医疗问答系统
- 通用模型: "感冒了怎么办？" → "多喝水，休息"
- 微调后: "感冒了怎么办？" → "建议检查是否为流感，需要化验血常规，根据结果判断是否需要抗病毒药物..."

**场景2**: 法律文书生成
- 通用模型: 格式不规范，术语不专业
- 微调后: 严格遵守法律文书格式，使用专业术语
